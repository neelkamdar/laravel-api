Manage Order

How can the User purchase products?

When the user clicks Add for a product, it goes into the cart. To checkout, hit Checkout and the user will be taken to the checkout page.

To make the POS button visible, an Admin must grant permission to the specific role.

Shipping Address

To set the Shipping Address for the customer, choose the Add New option if it's a new address or select existing ones.

To manage addresses, go to My Account → Saved Address where you can edit or delete them easily.

Billing Address

To set the Billing Address for the customer, choose the Add New option if it's a new address or select existing ones.

To manage addresses, go to My Account → Saved Address where you can edit or delete them easily.

Delivery Options

You can choose any of the two delivery options:

  1. Standard Delivery
  2. Same Day Delivery (Select your preferred time slot)

Payment Options

Also, select the Payment method as well to checkout.

Note: Here, the admin has the authority to change the delivery as well as the payment options.

Please ensure all fields are correctly filled before confirming your order.

You have the option to use points and wallet balance by checking the respective boxes.

Coupon codes can be applied at checkout

To complete your order, simply click on Place Order. You will then be redirected to the Order Details page.
