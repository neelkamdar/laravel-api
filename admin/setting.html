Other Configuration?

Activation

In the Activation tab, you can set various key parameters for your multi-vendor platform, including multi-vendor support, product auto-approval, wallet features, point systems, coupon utilization, and store auto-approval, with detailed guidance provided for each.

Wallet Points

A digital wallet is a form of virtual money that customers can spend on purchases.

Gmail Configuration

In the Mailer Tab, you can select any of the 3 options:

  1. Sendmail
  2. SMTP
  3. Mailgun

Please review the document and complete the mail configuration form.

https://docs.cpanel.net/knowledge-base/email/how-to-create-and-connect-to-an-email-account/

Vendor Commissions

In the Vendor Commissions tab, you can effectively manage the status of different features, specify the minimum withdrawal amount, establish commission rates, and set up category-based commissions. This helps ensure your website runs smoothly with clear instructions.

Refund

Specify the timeframe during which users are allowed to request refunds for items that have been delivered. If this field is empty or not set, the refund request functionality will be turned off.
