Home Page

How to make changes in the Home Page?

To change your website's theme, follow these simple steps:

  1. Visit the Store Front and go to the Themes section.
  2. Choose and activate the theme you prefer.
  3. To make changes, simply hover over the theme and click on the edit option.

Add the necessary details to the Home Banner.

Select link: When you choose a link from the dropdown menu, here's what happens:

    • If you pick "Product," it takes you to the chosen product.
    • If you select "Collection," it leads you to the chosen category.
    • For "External URL," it redirects you to a web link you provided in the format "http."
