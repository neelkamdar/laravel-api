Add Product

How to Add the Products?

Accessing the Product Option within the admin dashboard is a straightforward process.

Once you navigate to this section, you'll find the "Add Product" option readily accessible. The administrator's next step is to complete all the necessary information within the designated tabs.

Product Type :

3 type of product:

  1. Physical Product
  2. Digital Product
  3. External/Affiliate Product

*The product type field displays only the create a product

General Tab

Within the General Tab, specific details need to be provided:

Store :

You have the option to choose a store associated with the product from the provided dropdown menu. Furthermore, you can add a new store or make modifications to existing ones using this.

How to add store ?

Product Name :

In this tab, you can input the name of the product.

Short Description :

Here, you can provide a brief and concise description of the product. The maximum length should be 300 characters.

Description :

Here, you can provide a detailed description of your product.

Tax :

From the drop-down menu, choose the tax that applies to you.

Product Image Tab

In the Image Tab, the following details will have to be added:

Thumbnail :

You can insert a thumbnail image here.

Images :

You can upload up to 5 images in any format, but we recommend using the PNG format.

Size Chart :

The image for the size chart can be added depending upon the product.

Watermark :

Enable this to set watermark in images.

Watermark Position :

You can select watermark position link: Top left, Top, Top Right, Left, Center, Right, Bottom, Bottom Left, and Bottom Right.

Watermark Image :

You can insert a watermark image here.

Inventory Tab

In the Inventory Tab, the following details will have to be added:

Type :

2 type of product type Simple Product and Variable Product.

Simple

Stock Status :

You can indicate whether a product is available or not using a drop-down menu. The two options are:

  1. In Stock (Available)
  2. Out of Stock (Not Available)
SKU :

When adding a product to your website, you need to create a unique code for it.

Stock Quantity :

The amount of the stock has to be added.

Price :

You can enter the product's price here. The currency we use by default is the dollar sign ($). If you want to use a different currency, kindly go to the Settings. Only simple product types are displayed in this field.

Discount :

Enter the percentage discount you want to give for a product here. Only simple product types are displayed in this field.

Sales Price :

After you enter the discount percentage, the website will calculate and show you the discounted price. Only simple product types are displayed in this field.

Wholesale Price Type :

2 type of Wholesale Price Type Fixed and Percentage. Only simple product types are displayed in this field.

Enter specify the minimum and maximum quantity requirements. Additionally, if selecting fixed provide the price; if selecting percentage specify the percentage.

Variable Product

Select the product type variable product to enable the Variation tab.

Attributes :

Select the related attribute from the drop down menu.

Value :

The value can be selected from the drop down menu. The following details will have to be added in the "Variants" Section.

    • Multiple attributes can also be added using the Add Variant Tab.
Name :

Enter variation product name.

Price :

You can enter the product's price here. The currency we use by default is the dollar sign ($). If you want to use a different currency, kindly go to the Settings. Only simple product types are displayed in this field.

Discount :

Enter the percentage discount you want to give for a product here. Only simple product types are displayed in this field.

Sales Price :

After you enter the discount percentage, the website will calculate and show you the discounted price. Only simple product types are displayed in this field.

Stock Quantity :

The count of the stock has to be added.

SKU :

When adding a variation product to your website, you need to create a unique code for it.

Stock Status :

You can indicate whether a variation product is available or not using a drop-down menu. The two options are:

  1. In Stock (Available)
  2. Out of Stock (Not Available)
Image :

You can insert a variation image here.

Status :

Enable this filed to user purchase variation product.

Setup Tab

In the Setup Tab, the following details will have to be added:

Sales Status :

If you leave it turned on, a little label called 'Sale' will be included with the product information.

Sale Start Date :

You can quickly insert this date using the calendar.

Sale End Date :

You can quickly insert this date using the calendar.

Unit :

You need to include the appropriate unit for the product. Please indicate the measurement unit, such as 10 pieces, 1 kilogram, 1 litre, and so on.

Tags :
  1. Select the tags that describe your product from the drop-down list.
  2. Selecting relevant tags will help users to find your product more easily when they are searching for products on your website.
  3. The additional tags can be added using the following path: Products→ Tags → Add Tags
Categories :
  1. Select the categories that describe your product from the drop-down list.
  2. Selecting relevant categories will help users to find your product more easily when they are searching for products on your website.
  3. The additional tags can be added using the following path: Products→ Categories → Add Categories
Random Related Product :
  1. If you enable this option, the website will randomly select 6 products to show to visitors.
  2. Then, you can select the number of products to be displayed from the drop down menu of "Cross Sell Products".
Related Products: :

Choose up to 6 products that are related to the current product being viewed, and display them to the customer.

Cross Sell Products :
  1. To effectively display cross-selling products on your website, choose a maximum of 3 products.To effectively display cross-selling products on your website, choose a maximum of 3 products.
  2. This will help to avoid cluttering your page and overwhelming your visitors.

SEO

In the SEO, the following details will have to be added:

Meta Title :

Enter the MEta Title here.

Meta Description :

The meta description can be added here which is the text that appears below the Meta title in your search result listing.

Meta Image :

You can easily upload the image that will be shown on social media when your website is shared. We recommend using the PNG format, as it is a high-quality image format that is supported by most social media platforms.

Shipping & Tax

The Shipping & Tax Tab should include the following information:

Free Shipping :

Enable this to offer free shipping for a product

Weight :

The respective weight of the product will have to be added. The weight of the product must be specified in "Grams".

Estimated Delivery Text :

Enter the relevant text. Eg: Your order will arrive within 5 to 10 days.

Return :

To make the product eligible for returns.

Return Policy Text :

Enter the relevant text. Eg: Returns are easy with our hassle-free policy.

Status Tab

The Status Tab allows you to enable the following:

Featured :

To display a Featured tag on the product.

Safe Checkout :

A safe checkout image will appear on the product page. Modify the image in the theme options.

Secure Checkout :

A secure checkout image will appear on the product page. Modify the image in the theme options.

Social Share :

To allow users to share the product on social media platforms.

Encourage Order :

A random order count between 1 and 100 will be displayed to motivate user purchases.

Encourage View :

This feature encourages users to view products by presenting engaging content or prompts.

Trending :

To display the product as a trending one.

Status :

Enable this filed to show product on Frontend.
